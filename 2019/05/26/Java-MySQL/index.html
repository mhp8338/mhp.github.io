MySQL 数据库的简介

1.1 MySQL 数据库的简介和安装

1.1.1 什么是数据库？

数据库是按数据结构来组织、存储和管理数据的仓库

1.1.1 常见数据库

(甲骨文)Oracle、(IBM)DB2、(微软)SQL Server、Postgre SQL、MySQL。
MySQL 是典型的关系型数据管理系统

1.1.2 相关术语

数据库系统Database System简称 DBS，包含以下部分：

  • 数据库Database
  • 数据库管理系统Database Manage SystemDBMS
  • 应用开发工具
  • 管理员及用户

1.1.3 SQL 语言

SQL(结构化查询语言 Structured Query Language)

  • DDL:数据定义语言
  • DML:数据操控语言
    • 增
    • 删
    • 改
  • DQL:数据查询语言
    • 查询用户
  • DCL:数据控制语言
    • 控制用户权限

1.2 MySQL 相关操作

1.2.1 配置文件

  • 隐藏文件
  • my.conf

1.2.2 登陆/退出 MySQL

登陆：

  • mysql -uroot -p
  • mysql -uroot -p0623

退出:

  • exit
  • quit

版本：

  • mysql -V
  • mysql --version

登陆信息中需要掌握：

  • 登陆过后命令行的结束符使用;或者是\g
  • 可以通过help、\h或?配合关键字查看帮助手册
  • \c可以取消当前命令的执行：select user() \c

1.2.3 常用的SQL语句

SELECT USER():得到登陆的用户
SELECT VERSION():得到mysql的版本信息
SELECT NOW():得到当前的日期时间
SELECT DATABASE():得到当前打开的数据

1.2.4 SQL语句的语法规范

  • 常用关键字需要大写，库名、表名和字段名称等需要小写
  • 支持拆行操作，但不能把完整的单词拆开
  • 数据库名称，表名称和字段名称不使用MySQL的保留字，如果必须使用''将其括起来
