（一）初始 vim

1.1 vim 编辑插入模式

insert->normal:esc
normal->insert:

  • i:insert光标前插入
  • a:append光标后插入
  • o:open a line below当前行的下一行插入
  • A:append after line当前行最后插入
  • I:insert before line当前行最前方插入
  • O:append a line above当前行的上一行插入

保存和退出:wq

1.2 vim 的多种模式

  • normal:
    • 各种命令操作和移动
    • 大部分情况下在浏览而不是编辑
  • insert：
  • command:
    • 保存和关闭:wq
    • 分屏：:vs(竖分屏 vertical screen) :sp(横分屏 split)
    • 全局替换：:s/foo/bar/g->:s/python/java/g
  • visual:
    • Normal 模式下v进入 visual
    • 使用V选择行
    • ctrl+v块状选择文本

1.3 vim 插入模式技巧

ctrl+h:删除上一个字符
ctrl+w:删除上一个单词
ctrl+u:删除当前行
ctrl+[:切换 esc
gi:快速跳转到最后一次编辑的地方进入插入模式

1.4 vim 快速移动

w/W移动到下一个 word/WORD 开头
e/E移动到下一个 word/WORD 结尾
b/B移动到上一个 word/WORD 开头
WORD 以空白符分割的单词，word 以非空白符分割的单词
行间搜索移动：

  • f/F跳转到目标字符(前面搜索/后边搜索)
  • ;/,搜寻到下一个/上一个
    水平移动:
  • 0移动到行首，^移动到第一非空白字符=0+w
  • $移动到行尾，g_移动到行尾非空白字符

垂直移动：

  • (和)句子之间移动
  • {和}段落之间移动

页面移动：

  • gg/G移动到文件的开头和结尾，ctrl+o快速返回
  • H/M/L跳转到屏幕的开头(Head)、中间(Middle)和结尾(Lower)
  • ctrl+u/f上下翻页(upword,forword)
  • zz把屏幕置于中间

1.5 vim的增删改查

1.5.1 vim增加字符

  • 增加字符就是插入模式
  • iao
  • IAO

    1.5.2 vim删减

  • x也可以删除
  • daw删除单词
  • diw不删除空格
  • dd删除行
  • dt)删除右括号里的内容
  • d配合visual模式的使用

    1.5.3 vim快速修改

    常用有三个：
  • c(change):配合文本删除字符进入插入模式c+(a)+w C ctg
  • r(replace):快速修改
  • s(substitude):删除当前字符进入插入模式

    1.5.4 vim快速查询

    查询单词也是一种常用操作：
  • 使用\或者?进行前向或者反向操作
  • 使用n或者N跳转下一个或者上一个匹配
  • 使用*或者#进行当前单词的前向和后向匹配

    vimrc

vimrc 的几点说明：

  • vimrc=vim run command
  • 系统级 vimrc 和用户级 vimrc
    • 系统级：所用用户都生效
    • 用户级：不同用户命令生效状态不同
    • 每一行作为一个命令而执行

vim删除

  • d$删除光标后的改行部分
