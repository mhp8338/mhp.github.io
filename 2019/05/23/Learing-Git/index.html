Learing Git

Posted on 2019-05-23 | In Git |

创建 Git 版本库

版本库就是一个目录，在该目录下所有文件都受 Git 的管理，Git 可以跟踪每个文件的修改和删除，任何时刻都可以追踪和还原。
具体过程如下：

  • 创建空目录
      mkdir learngit
  cd learngit
  • 将该目录变成 Git 所管理的仓库
    git init
  • 编写 readme.txt
  • 把文件添加仓库缓存区stage
    git add readme.txt
  • 提交更改，把缓存区内容提交至当前分支master
    git commit -m "wrote a readme file"

修改 git 文件

  • 修改 readme.txt 文件
  • 查看修改修改状态，但此时还未提交修改。并且只能显示已经修改，并未显示具体修改的内容
    git status
  • 查看具体修改的内容
    git diff
  • 添加
    git add readme.txt
git commit -m "add distritbuted"

版本回退

  • 查看历史记录
    git log
git log --pretty=oneline
  • 回退版本
    git reset --hard Head^
git reset --hard Head~100
  • 还原版本
    git reset --hard {commit id}
  • 查看文件
    cat readme.txt

  • 查找之前每一次命令

    git reflog

    工作区和暂存区

  • 提交缓存区

  • 提交分支

git管理修改，并非管理文件。
第一次修改 -> git add -> 第二次修改 -> git add -> git commit
